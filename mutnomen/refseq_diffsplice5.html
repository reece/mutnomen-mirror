Exon larger at 5' end

Detailed expression analysis of this fictive gene showed that a minor fraction of the transcripts (<5%) contained an alternatively spliced exon 54, extending it with the last 27 nucleotides of intron 53. Based on a coding DNA reference sequence, numbering of these differentially spliced nucleotides is straightforward, i.e. as for the intronic nucleotides c.9661-27 to c.9661-1. The encoded amino acids can best be numbered p.Val3221-9 to p.Gln3221-1; this number clearly indicates their position in relation to the other amino acids with the '-' number indicating a location in an intron (see below).
