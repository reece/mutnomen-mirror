Version list


Last modified March 16, 2013  -  Version 2.121101

This page lists the latest changes in the HGVS recommendations for the description of sequence changes and shows the latest version number. For details, see Versioning.

New versions

  • date 2012-11-01
    Variants affecting translation termination      -  at protein level, variants that replace the translation termination codon but do not encounter a new stop codon in the new reading frame are described using the format "p.*321Argext*?", frame shift variants with the same effect are described using the format "p.Ile321Argfs*?" (see Protein descriptions)
  • date 2012-08-31
    • Protein description between brackets  -  brackets can be omitted when there is sufficient experimental proof from other studies.
    • Variants affecting translation initiation  -  at protein level, variants that generate a new upstream translation initiation codons are described using the format "p.Met1ext-5" (see Discussion).
